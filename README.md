# Sensor-Telemetry
SensorTelemetry — это консольное приложение на C++, предназначенное для генерации, валидации и анализа данных датчиков в формате JSON. Проект включает функционал:

Генерация JSON файлов с корректными и специально искажёнными данными для тестирования;

Валидация структуры и содержимого JSON с выводом информативных сообщений об ошибках;

Анализ данных выбранного датчика: вычисление минимального, максимального, среднего и медианного значения;

Скользящее окно для поиска аномалий;

Измерение времени выполнения критических операций (вычисление медианы, скользящее окно);

Удобный CLI для управления генерацией, валидацией и анализом.

Проект позволяет создавать до 100 000 JSON файлов с тестовыми данными, включая варианты с пропущенными или некорректными значениями, что обеспечивает надёжное тестирование системы.
