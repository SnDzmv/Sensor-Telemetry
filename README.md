# SensorTelemetry

**SensorTelemetry** — консольное приложение на C++ для генерации, валидации и анализа JSON-файлов с данными сенсоров.  
Программа позволяет:

- Генерировать JSON-файлы (корректные и с ошибками)  
- Валидировать данные (`ts`, `value`)  
- Выполнять статистический анализ: min, max, average, median  
- Обнаруживать аномалии с помощью скользящего окна  
- Измерять время выполнения вычислений (benchmarks)

---
##  Запуск программы

1. Скачай репозиторий и перейди в папку `release/`.  
2. Дважды кликни на `SensorTelemetry.exe`.

---
После запуска появится меню:
---

SENSOR TELEMETRY


Создайте JSON файл с названием: data_{number}.json
Исчисление файлов начинается с нуля
Выберите действие
1) Замер мин, макс, сред значения, медианы и нахождения аномалий
2) Режим отладки
3) Выйти из программы
---
Для анализа данных, вставьте свои JSON файлы в папку `release/`.
Пример JSON файлов находятся в `data/`.

---
Демонстрационный сценарий

- Генерация JSON файлов (Режим отладки):

- Выбираем 2) режим отладки → 2) Сгенерировать JSON файлы без ошибок

- Вводим количество файлов, например 5

- Создаются файлы data_0.json, data_1.json … data_4.json

- Валидация файлов (Замер мин, макс, сред значения, медианы и нахождения аномалий):

- Выбираем 1) Замер мин, макс, сред значения, медианы и нахождения аномалий

- Вводим количество файлов для проверки (5)

- Программа проверяет дубликаты ts, корректность value и выводит результат

Анализ данных сенсора:

- Вводим имя сенсора: s1

- Вводим размер скользящего окна, например 3

Программа выводит статистику и аномалии:
